# Sofia Lindelow Portfolio

Static Netlify-ready version of Sofia Lindelow's portfolio.

## Update flow

1. Edit `index.html`.
2. Commit the change to `main`.
3. Push to GitHub.
4. Netlify deploys automatically from the connected GitHub repo.

## Netlify settings

- Build command: leave empty
- Publish directory: `.`
- Production branch: `main`
